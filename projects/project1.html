HOW TO MAKE A SIMPLE WEBSITE

THIS IS A WEBSITE TUTORIAL FOR BEGINERS

Making a simple website is a bit easy if you use the right coding environment.There are different ways in making a WEBSITE but to make it easy for beginers it is divided into just 4 steps.

  • step1

    • The First step in making/designing a website is too download a suitable coding environment an Example of web developer is visual studio code this can be downloaded through the internet or directly from the VISUAL STUDIO WEBSITE

    visual studio code link

  • step2

    • The Second step after downloading V.S(VISUAL STUDIO) code is to install it into your desktop or laptop after following the instructions on the computer you click install

  • step3

    • The third step is to open V.S code and install some extension the first Extension you will be installing is:

  • LIVE SERVER

    • This extension creates a local server and allow you to see the changes in real time.

    the next extension to be installed is:

  • JAVA SCRIPT(ES6)CODE SNIPPETS

    • This will help you in your javascript coding as javascript will be the main part of your web development. The last extension you would be installing is BOOTSTRAP then you are done with the V.S CODE SETUP

  • step4

    • The last step is to create a project and in that project create an HTML file with the name index.html

    There is something in V.S CODE that is called EMMET,it Basically allows you to write code without actually writing it.For example if you write an exclamation mark you will see that the EMMET has popped up,it is suggesting to use the code. So you can just click on it ,you will see the basic structure of the HTML page,this shows that our web development environment is ready on V.S code and we can start developing our websites
